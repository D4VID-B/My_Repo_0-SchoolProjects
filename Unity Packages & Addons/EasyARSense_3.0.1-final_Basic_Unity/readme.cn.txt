本包中包含EasyAR SDK Basic的Unity样例。
请阅读EasyAR网站上文档来了解应该如何来使用这些样例：http://www.easyar.cn/view/support.html

样例:

  HelloAR_BackAndFrontCamera
    - 演示手机相机前后置摄像头来回切换

  HelloAR_Coloring3D
    - 演示如何创建“AR涂涂乐”，使绘图书中的图像实时“转换”成3D

  HelloAR_ImageTarget
    - 演示创建target的不同方法
    - 演示动态创建target

  HelloAR_ImageTarget_Cloud
    - 演示如何使用云识别

  HelloAR_ImageTarget_Video
    - 演示如何使用EasyAR加载并在target上播放视频

  HelloAR_ImageTracking_ImageTargetData
    - 演示通过ImageTargetData创建target的方法

  HelloAR_Multi_ImageTarget
    - 演示如何使用一个tracker同时跟踪多个目标

  HelloAR_SurfaceTracking
    - 演示如何使用表面跟踪

  HelloAR_TargetOnTheFly
    - 演示如何直接从相机图像中实时创建target并load到tracker中

  HelloAR_TransparentVideo
    - 演示如何使用Unity Video Player加载视频，用自定义shader实现透明视频播放

注意：Unity Videoplayer 在某些Unity版本上播放流媒体视频会有卡顿，你可以参考以下链接解决方案来解决这个问题
https://forum.unity.com/threads/unity-video-player-and-android-laggy.504570/

图片Markers:
  你可以在每个sample里面名字为StreamingAssets的文件夹中找到样例中使用的图片marker
